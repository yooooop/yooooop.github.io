Fight Night

Unreal Engine 5 | C++ | Multiplayer | AI

Two-player cooperative fighting game where each player controls part of a single fighter. Built to explore multiplayer architecture, replicated gameplay systems, and AI-driven combat.

Video Demo | GitHub

Core Concept

Fight Night splits control of a single fighter between two players: movement (legs) and combat (arms). Success depends on coordination, timing, and shared decision-making.

Multiplayer System

  • Server-authoritative architecture
  • Replicated movement, combat, and health systems
  • Two independent input streams controlling one pawn
  • Validated attack and damage execution on server

AI System

  • Utility-based decision system
  • Dynamic evaluation of distance and combat state
  • Shared combat logic with player systems

Architecture

  • Separation of movement and combat modules
  • Event-driven combat pipeline
  • Replication-driven gameplay state sync
  • Shared systems between AI and players
