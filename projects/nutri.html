Nutri

Kotlin | AWS | Android | ChatGPT API

Mobile nutrition-tracking application that combines barcode scanning, cloud-based services, and AI-powered dietary analysis.

Video Demo

Overview

Nutri is an Android application designed to simplify nutritional tracking. Users can scan food products, retrieve nutritional information, and receive dietary insights generated through AI-powered analysis.

Architecture

  • Android application built in Kotlin
  • AWS-backed cloud infrastructure
  • Barcode scanning workflow for food lookup
  • Serverless backend services
  • AI-driven dietary recommendations

Technical Contributions

  • Designed and implemented application architecture
  • Integrated AWS cloud services for backend functionality
  • Implemented barcode scanning workflow
  • Connected ChatGPT API for dietary analysis
  • Applied Observer and Factory design patterns

Technical Challenges

One challenge was designing a system that could combine user-generated food data, cloud-hosted services, and AI-generated recommendations while maintaining a responsive mobile experience.
