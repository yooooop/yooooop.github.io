Rhythm

Unity | C# | Gameplay Programming

Rhythm-based boss-fight game created during a week-long game jam, focused on gameplay systems and UI implementation.

Video Demo

Overview

Rhythm is a fast-paced action game where player actions are synchronized with music and boss attack patterns. The project was completed under strict time constraints during a game jam.

Contributions

  • Implemented gameplay systems in Unity using C#
  • Developed UI elements and gameplay feedback systems
  • Collaborated within a multidisciplinary team
  • Rapidly iterated on mechanics during development

Game Jam Constraints

The project was completed within one week, requiring rapid prototyping, constant iteration, and prioritization of core gameplay features.
